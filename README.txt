VIDEOMAKER CIRO V2

Esta versión SÍ genera video descargable.

Funciones:
- Crear escenas desde texto.
- Una línea = una escena.
- Subir imágenes como fondos.
- Formato vertical, horizontal o cuadrado.
- Vista previa.
- Crear video descargable en formato WEBM.
- Funciona en navegador moderno.
- Puede publicarse en GitHub Pages.

Nota:
No es IA generativa tipo Sora. Es un generador de video educativo con texto, escenas, fondos e imágenes.
En iPhone/iPad, el formato WEBM puede abrirse mejor desde Archivos o apps compatibles.
